lab1/
├── pipeline.sh              # Главный скрипт-оркестратор

├── data_preprocessing.py    # Этап 2: Предобработка
├── model_preparation.py     # Этап 3: Обучение модели

│   └── y_scaled.csv

## Требования
- Python 3.8+
- Linux / WSL 
- pip, python3-venv

## Запуск
cd lab1
chmod +x pipeline.sh
bash pipeline.sh

## Данные
Тип задачи - Бинарная классификация
Количество объектов - 1000

- Данные создаются синтетически через sklearn.datasets.make_classification
- 5 признаков: 3 информативных, 1 зависимый, 1 шумовой
- Добавлен гауссовский шум

## Модели

Алгоритм - Логистическая регрессия
Библиотека - scikit-learn

Предобработка:
StandardScaler - нормализация признаков 
Скалер обучается только на train-данных (предотвращает утечку)

Метрика:
Accuracy - доля правильных предсказаний на тестовой выборке

Типичный результат: 
Model test accuracy is: 0.945

## Этапы

data_creation.py - Генерация данных, разделение на train/test, сохранение в CSV

data_preprocessing.py - Нормализация признаков через StandardScaler

model_preparation.py - Обучение логистической регрессии, сохранение в pickle

model_testing.py - Загрузка модели, предсказание, вывод accuracy

## pipeline.sh

Скрипт автоматически:
- Проверяет наличие Python 3 и pip
- Проверяет/создаёт venv/
- Устанавливает зависимости (numpy, pandas, scikit-learn)
- Последовательно запускает 4 этапа конвейера
- Выводит итоговую метрику или ошибку

Особенности:
- Виртуальное окружение создаётся один раз и переиспользуется
- При ошибке выводится сообщение с инструкцией
